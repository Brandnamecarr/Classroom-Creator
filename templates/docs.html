Getting Started:

1. Using the checkbox, choose if you will use Canvas (default) or a CSV file for your class list.
2. If using Canvas, click the box on the left to sign in. If using a CSV file, use the box on the left to upload.
3. Use the box on the right to sign into your Google Classroom
4. Click the migrate button. Classroom Creator automatically creates your Google Classroom and reports changes.

FAQ:

1. How should I format my CSV file?
  See the sample CSV file
2. Can I add new Google Classrooms if I already have existing classes?
  Yes! Classroom Creator will automatically check for existing classes and record a log of changes.
3. How do I know my classroom data is safe?
  Classroom Creator does not directly access the data, instead using Google and Canvas API's to transfer data.
4. How do I contact Classroom Creator with questions or support requests?
  Contact by email at UFClassroomCreator@gmail.com. Please allow 48 hours for a response.
